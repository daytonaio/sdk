title: File System
---

File system operations for Daytona workspaces.

Provides comprehensive functionality for managing files and directories
within a workspace, including creation, deletion, modification, and search operations.

### Examples

```ts
Basic file operations
// Create a directory and write a file
await workspace.fs.createFolder('/app/src', '755');
await workspace.fs.uploadFile('/app/src/index.ts', file);
```

```ts
File search and manipulation
// Search for files and replace content
const matches = await workspace.fs.searchFiles('/app', '*.ts');
await workspace.fs.replaceInFiles(
  matches.map(m => m.file),
  'oldValue',
  'newValue'
);
```

| ------ | ------ | ------ |
| `path` | `string` | Directory to search in |
